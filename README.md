# Calculadora Básica

Este proyecto es una calculadora básica en consola que permite realizar operaciones de suma, resta, multiplicación y división. Fue desarrollado como parte de un trabajo práctico para la asignatura de [Programación 1] en el Instituto [Beppo Levi].

## Características

- Realiza operaciones básicas: suma, resta, multiplicación y división.
- Guarda un historial de las operaciones realizadas.
- Permite consultar cuál fue la operación más realizada.

## Requisitos

- .NET Framework 4.8 o superior.
- IDE compatible con C# (Visual Studio, JetBrains Rider, etc.).
